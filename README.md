# 📊 Retail Sales Analysis using Python

A comprehensive Retail Sales Analysis project built with **Python, NumPy, Pandas, and Matplotlib** to analyze retail transaction data, perform exploratory data analysis (EDA), and generate meaningful business insights through statistical analysis and data visualization.

---

## 📌 Project Overview

This project analyzes a retail sales dataset to understand customer purchasing patterns, product performance, sales trends, and business metrics. It demonstrates the use of Python libraries for data manipulation, statistical analysis, and visualization.

The project covers:

- Data Loading
- Data Cleaning
- NumPy Operations
- Statistical Analysis
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Correlation Analysis
- Data Visualization
- Business Insights

---

## 🚀 Features

- Import and process retail sales data from Excel
- Convert and analyze date information
- Perform NumPy array operations
- Mathematical calculations
- Statistical analysis
- Logical and comparison operations
- Indexing and slicing
- Array reshaping
- Axis-wise operations
- Data filtering and sorting
- GroupBy analysis
- Feature engineering
- Correlation analysis
- Interactive business insights
- Data visualization using Matplotlib
